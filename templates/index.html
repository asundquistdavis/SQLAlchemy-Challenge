Hawaii Weather API

The following urls are enpoints for this api.

URL Function
/api/v1.0/precipitation Lists all available precipitation data by date.
/api/v1.0/stations Lists all the station from the dataset.
/api/v1.0/tobs/<station> Lists temperature observations for the previous year for the given <station>.
/api/v1.0/<start>/<end> Lists the min, max and average temperature in the specified range.
